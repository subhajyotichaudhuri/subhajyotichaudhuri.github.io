<p> GaussKIT provides powerful data-analysis, post-processing, visualization, and interfacing with electronic structure codes that use gaussian basis sets. Most of the tools are fully compatible with the latest builds of QChem and Orca. NWChem implementations coming soon.</p>

<p>Huge shoutout to the members of the community for reporting bugs in 1.0.2. The Github repository is currently offline while I fix the compatibility issues. </p>

`Update: Git currently offline. Stable build v 1.0.3 releasing in January 2024`
